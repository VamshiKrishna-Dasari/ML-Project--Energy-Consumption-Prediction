# Energy Consumption Prediction using KNN and ANN

## 📌 Project Overview
Energy consumption forecasting is important for improving energy efficiency and managing resources effectively.  
This project predicts energy consumption using two machine learning models:

- K-Nearest Neighbors (KNN)
- Artificial Neural Networks (ANN)

The goal is to analyze historical energy data and predict future energy usage patterns.

---

## 🎯 Objectives
- Analyze energy consumption data
- Build prediction models using KNN and ANN
- Compare the performance of both models
- Provide insights for energy optimization

---

## 🛠 Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- TensorFlow / Keras
- Matplotlib
- Jupyter Notebook

---

## 📊 Dataset
The dataset contains historical records related to energy consumption and environmental factors.

Example features:
- Temperature
- Humidity
- Appliance usage
- Lighting usage
- Time-related attributes

Data preprocessing steps include:
- Handling missing values
- Data normalization
- Splitting data into training and testing sets

---

## ⚙️ Methodology

### 1. Data Preprocessing
- Clean the dataset
- Normalize the data
- Split into training and testing sets

### 2. K-Nearest Neighbors (KNN)
- Uses similarity between data points
- Predicts energy consumption based on nearest neighbors

### 3. Artificial Neural Network (ANN)
- Multi-layer neural network model
- Learns complex patterns in energy data

### 4. Model Evaluation
Models are evaluated using:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

---

## 📈 Results
- Both KNN and ANN models were used to predict energy consumption.
- ANN captured complex patterns better.
- KNN provided simpler and interpretable predictions.
